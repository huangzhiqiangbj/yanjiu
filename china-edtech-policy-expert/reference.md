# 中国教育科技政策参考

正式任务若需要 `政策原文`、`官方链接`、`逐年比较` 或 `精确表述`，先读 [policy-corpus.md](policy-corpus.md)。本文件侧重阅读框架、主题图谱和解释方法，不替代原文库。

## 0. 必学政策母体与阅读顺序
遇到中国教育科技政策问题时，先在这些 `政策母体` 中定位总逻辑，再下钻到专项文件和地方细则。

| 政策母体 | 重点读什么 |
| --- | --- |
| `党的二十大报告` | 读 `教育、科技、人才` 一体部署，理解教育为何被放进现代化建设和国家竞争力框架 |
| `党的二十届三中全会《决定》` | 读教育综合改革、教育数字化、科技教育与人文教育协同、学科专业调整机制 |
| `近五年政府工作报告` | 读年度教育关键词怎么变、排序怎么变、哪些提法从工作重点上升为国家主线 |
| `《教育强国建设规划纲要（2024-2035年）》` | 读教育强国的目标体系、阶段目标、基础教育体系、教师体系、科技支撑体系 |
| `《加快建设教育强国三年行动计划（2025-2027年）》` | 读纲要如何转成近中期任务、改革试点和推进节奏 |
| `国家教育数字化战略行动` | 读教育数字化的核心平台、公共服务逻辑、资源中心和终身学习支撑 |
| `教育部等九部门关于加快推进教育数字化的意见` | 读 `集成化 + 智能化 + 国际化`，尤其是 AI 融入教育教学全要素全过程 |
| `教育部办公厅关于加强中小学人工智能教育的通知` | 读中小学 AI 教育的学段分层、课程体系、资源平台、师资和基地建设 |
| `人工智能助推教师队伍建设行动` | 读 AI 如何进入教师培养、教师研修、智能助手和教师治理 |
| `“十五五”规划纲要` | 读未来五年教育科技人才一体发展、高质量教育体系、基础教育扩优提质和前沿学科布局 |

### 0.1 二十大以来教育政策总主线
把零散政策放回下面这些主线里，才算读深：
- `教育、科技、人才一体统筹推进`
- `从办好人民满意的教育走向教育强国建设`
- `从教育公平底线保障走向高质量教育体系建设`
- `从资源数字化走向教育数字化公共能力建设`
- `从技术试用走向 AI+教育全要素全过程`
- `从单点场景创新走向价值导向、安全治理、未成年人保护同步推进`

### 0.2 近五年政府工作报告教育表述迁移
读政府工作报告，重点不是摘抄，而是看重心如何迁移。可先按下列线索理解，再以各年正式全文为准核对。

- `2022`：重心仍明显落在 `双减`、义务教育薄弱环节、未成年人保护、心理健康、在线教育、终身学习等基础治理与民生面向。
- `2023`：在总结过去五年教育投入和公平成效的基础上，开始更突出 `义务教育优质均衡发展和城乡一体化`、职业教育和高等教育创新。
- `2024`：`教育强国、科技强国、人才强国建设一体统筹推进` 成为更强主线，基础教育扩优提质、数字教育、高素质教师队伍被进一步前置。
- `2025`：教育强国从中长期纲要转向近中期实施抓手，出现 `教育强国建设三年行动计划`、`新时代立德树人工程`、`思政课一体化改革创新` 等更明确部署。
- `2026`：教育进一步进入 `十五五` 发展框架，`教育科技人才一体发展`、学科专业动态调整、重大工程、拔尖创新人才培养和教师队伍建设被放到更高位置。

一句话判断：
- 近五年政府工作报告中的教育表述，整体呈现出从 `公平与治理` 到 `高质量体系`，再到 `教育强国 + 教育科技人才一体 + 数字化与AI` 的持续抬升。

### 0.3 “十五五”问题的回答纪律
用户问 `十五五` 时，默认遵守以下规则：
- 优先核 `中国政府网` 的正式纲要文本和专题页，不用地方转述替代中央正式文本。
- 重点看 `教育科技人才一体发展`、`高质量教育体系`、`基础教育扩优提质`、`劳动年龄人口平均受教育年限`、`学科专业布局`、`人工智能等前沿领域急需学科专业`。
- 回答时要把 `十五五` 放回 `二十大 -> 二十届三中全会 -> 教育强国纲要 -> 政府工作报告` 这条链路里，不能孤立解读。

### 0.4 AI+教育的深层理解框架
回答 `AI+教育` 时，默认按这几个层次理解，而不是只谈产品提效：
- `国家战略层`：AI+教育服务于教育强国、科技强国、人才强国建设，一开始就是国家能力建设问题。
- `教育系统层`：AI 不只进入课堂工具，还进入课程、教材、教学、评价、教研、治理、平台、资源和终身学习体系。
- `学生发展层`：AI 教育既包括学生使用 AI，也包括培养人工智能素养、科学精神、创新能力和技术伦理。
- `教师发展层`：AI 应减轻教师重复劳动、提升研修与教研质量，但教师专业判断、价值引导和育人责任不能外包给模型。
- `公共服务层`：国家智慧教育平台、资源中心、教育数据中心等公共基础设施，是 AI+教育规模化落地的前提。
- `治理安全层`：价值导向、数据安全、算法安全、未成年人保护、校内准入和责任归属，是 AI+教育必须同时回答的问题。
- `产业边界层`：企业是能力供给者和服务协作者，不是教育目标和评价标准的替代制定者。

### 0.5 常见表述的真实含义
看到这些政策表述时，不要停留在字面：
- `教育强国`：不是泛口号，而是教育体系质量、人才供给、科技支撑、教师体系和国际竞争力的系统工程。
- `教育数字化`：不是简单上平台，而是教育公共服务、资源配置、数据治理和终身学习的基础设施建设。
- `全要素全过程`：不是多加几个 AI 工具，而是把 AI 融入课程、教材、教学、评价、治理和服务全链条。
- `智能向善`：不是软性口号，它直接对应价值导向、算法安全、内容安全和未成年人保护。
- `教育科技人才一体改革`：不是三个词并列，而是把教育供给、科技创新和人才培养放进同一国家竞争框架。

## 1. 官方来源优先级
解读中国教育科技政策时，默认按以下顺序取证：

1. `全国人大 / 中国政府网 / 国务院公报`
   - 适合确认 `法律`、`行政法规`、国务院层面的顶层文件
   - 常见站点：`npc.gov.cn`、`gov.cn`
2. `教育部及相关中央部门官网`
   - 适合确认教育系统专项政策、配套规则和解释口径
   - 常见站点：`moe.gov.cn`、`cac.gov.cn`、`miit.gov.cn`、`samr.gov.cn`
3. `省市政府 / 教育厅局 / 地方教育局官网`
   - 适合确认地方实施细则、试点方案、入校规则、区域推进节奏
4. `官方标准、答问、政策解读、新闻发布会通稿`
   - 适合补充 `怎么理解` 和 `怎么执行`
5. `权威研究机构或官媒解读`
   - 只能作为辅助，不替代原始政策文本
6. `普通媒体、行业文章、二手转述`
   - 只能用来发现线索，不能作为硬依据

默认规则：
- 先找原文，再看解读
- 先看上位规则，再看地方细则
- 先确认发文机关和文种，再判断约束力

## 2. 政策位阶速判
| 层级 | 常见形式 | 解读重点 |
| --- | --- | --- |
| 法律 | 法、修正案、决定 | 先判断刚性边界和基本义务 |
| 行政法规 | 条例、规定 | 看全国层面的具体规范与管理机制 |
| 部门规章 | 办法、规定、部令 | 看行业监管要求和可执行动作 |
| 规范性文件 | 意见、通知、实施方案、工作要点、指导意见 | 看对象、动作、约束词和是否有配套机制 |
| 地方性法规 / 地方政府规章 | 地方条例、地方办法 | 只在本地适用，不能外推全国 |
| 地方规范性文件 / 试点文件 | 实施细则、工作方案、试点通知 | 重点看本地执行抓手和范围 |
| 标准 / 指南 / 案例 | 标准、指南、典型案例、参考框架 | 常用于操作参照，通常不直接等于处罚依据 |

注意：
- 文件标题里的 `意见`、`方案`、`通知` 并不能单独决定约束力，必须结合 `发文机关` 和 `具体措辞` 判断。
- 同一主题下，`上位规则` 比 `地方细则` 更适合回答“能不能”，地方细则更适合回答“怎么做”。

## 3. 常见政策主题图谱
### 3.1 教育数字化与平台建设
重点看：
- 国家和地方的教育数字化部署
- 平台、资源、网络、终端、教育新基建
- 区域教育公共服务平台与学校接入机制

优先查看：
- `gov.cn`
- `moe.gov.cn`
- 各省教育厅、教育局官网

### 3.2 人工智能 + 教育与生成式AI应用
重点看：
- AI 在课堂、作业、评价、教师发展、学习支持中的边界
- 生成式AI是否进入学生端、教师端、校内平台
- AI 素养、人工智能课程、校内试点与治理要求

优先查看：
- `moe.gov.cn`
- `gov.cn`
- `cac.gov.cn`
- 地方教育主管部门的配套通知

### 3.3 双减、校外培训与类培训产品
重点看：
- 学科培训监管与变相培训边界
- 智能硬件、AI答疑、作业辅导产品是否踩到培训治理红线
- 节假日、收费、广告宣传、课程组织方式的监管要求

优先查看：
- `gov.cn`
- `moe.gov.cn`
- 地方政府和教育局官网

### 3.4 教育App入校与校园平台治理
重点看：
- 进校准入、备案、遴选、白名单、整治要求
- 数据采集范围、商业广告、诱导付费、家校群营销
- 学校统一管理与教师、家长使用边界

优先查看：
- `moe.gov.cn`
- 地方教育局官网
- 相关网信、市场监管部门通知

### 3.5 数据治理、个人信息与未成年人保护
重点看：
- 学生个人信息、学习行为数据、画像、算法推荐
- 校园平台的数据留存、共享、跨主体流转
- 未成年人网络保护、内容推荐、深度合成与生成式服务

优先查看：
- `npc.gov.cn`
- `gov.cn`
- `cac.gov.cn`
- `moe.gov.cn`

### 3.6 课程、教材与内容合规
重点看：
- 校内课程是否合规纳入
- 教材、数字资源、生成内容、价值导向与审核责任
- AI生成内容是否能直接作为教学内容或评价依据

优先查看：
- `moe.gov.cn`
- 地方教育厅局

### 3.7 教师数字素养与组织能力建设
重点看：
- 教师培训、能力标准、校本研修、区域师训
- 技术使用责任是否压给教师个人，还是由学校和区域统筹
- 人机协同与教师专业判断边界

优先查看：
- `moe.gov.cn`
- 地方教师发展、师训、教研相关文件

### 3.8 采购、试点与区域推进
重点看：
- 是否有公开试点、示范区、项目制推进、财政支持
- 采购主体、平台准入、评估验收、规模化条件
- 政策是鼓励探索，还是已形成常态部署要求

优先查看：
- `gov.cn`
- `moe.gov.cn`
- 地方政府、教育局、财政相关公开文件

### 3.9 高教与职教外延
当用户明确提到高教、职教、产教融合、就业导向时，再外延到：
- 课程体系、专业建设、实训平台、校企合作
- 科技成果转化、人工智能相关专业布局

## 4. 默认问题链
正式解读前，至少追问以下问题：
- 这份政策是 `上位规则`、`执行细则`、`试点文件` 还是 `倡议性部署`
- 它主要在管 `学校`、`培训机构`、`企业平台` 还是 `地方主管部门`
- 它解决的是 `校内教学` 问题、`校外培训` 问题、`数据治理` 问题，还是 `平台准入` 问题
- 哪些词是硬约束，哪些词是方向性表述
- 它是全国普遍适用，还是只在特定地区、特定阶段、特定项目内适用
- 这份文件与更高层级政策之间是什么关系
- 它对学校、区域、企业分别意味着什么
- 哪些地方可以明确判断，哪些地方仍需继续核原文或地方细则

## 6. 常用官方检索写法
优先检索 `官方站点 + 主题词`，不要先搜媒体稿：

- `site:moe.gov.cn 人工智能 教育`
- `site:gov.cn 教育 数字化`
- `site:moe.gov.cn 教育App 入校`
- `site:cac.gov.cn 未成年人 算法 推荐`
- `site:gov.cn 校外培训 智能`
- `site:moe.gov.cn 教师 数字素养`

如果结果很多，先加：
- 年份
- 具体学段
- 地区名
- 具体对象，例如 `学校`、`培训机构`、`平台`

## 7. 最容易出错的地方
- 把 `校外培训治理` 误套到所有校内数字化教学场景
- 把 `地方试点` 误写成全国统一要求
- 把 `宣传口径`、`案例稿`、`媒体稿` 当成政策依据
- 不区分 `学校责任` 与 `企业责任`
- 不区分 `教育主管部门文件` 与 `网信、市场监管、工信等跨部门规则`
- 只写“政策鼓励/支持”，却不说明 `鼓励谁`、`鼓励做什么`、`边界在哪`
