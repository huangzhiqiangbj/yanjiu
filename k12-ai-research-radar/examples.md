# Example Prompts

## 一、周扫描
```text
请按 k12-ai-research-radar 工作流，扫描最近 14 天基础教育 AI 新论文，重点看技术变革、政策影响、实践路径，先给候选清单，不要直接展开成长综述。
```

```text
请做一轮周扫描，主题是“生成式AI在学校治理中的新研究”，范围限基础教育，输出中文，给我 Top 5 候选论文和每篇入选理由。
```

## 二、月扫描
```text
请按 k12-ai-research-radar 的配套 agent workflow，扫描最近 30 天基础教育 AI 新论文，来源优先国际核心期刊、AIED/LAK/EDM、国内教育技术核心期刊和 UNESCO/OECD 等权威机构，输出候选清单 + 3 篇建议深读。
```

```text
请做一份“基础教育AI月度研究雷达”，重点不是教学设计，而是技术变革、政策影响和实践路径。请按高质量来源筛选，生成候选清单，并说明本期最值得关注的变化。
```

## 三、季度扫描
```text
请对最近 90 天的基础教育 AI 文献做季度扫描，聚焦政策影响与实践路径，输出中文。先给候选清单，再总结研究共识、研究分歧和下一季度值得继续跟踪的问题。
```

```text
请做一轮季度研究扫描，主题是“AI在基础教育中的治理与规模化实施”，生成 Top 10 候选文献，并标出哪些适合深读，哪些只适合背景跟踪。
```

## 四、专题扫描
```text
请按这个 skill 的扫描工作流，专门扫描“生成式AI与基础教育政策治理”方向最近 45 天的新文献。请分国际学术源、国内学术源、权威机构报告三类给候选清单。
```

```text
请扫描“基础教育AI实践路径”相关新文献，重点看学校或区域如何从试点走向常态化。不要把教学设计类小样本研究放在前面。
```

## 五、候选清单后续深读
```text
基于刚才那份候选清单，请从中选出最值得深读的 3 篇，分别输出论文卡，并说明它们对中国教育场景的迁移价值和边界。
```

```text
请基于这轮扫描结果，再提炼 5 条最值得长期跟踪的观点，每条都说明证据基础、成立条件和治理边界。
```


```text
扫描最近30天基础教育AI新论文，给候选清单。
```

```text
做一轮“生成式AI与学校治理”专题扫描。
```
